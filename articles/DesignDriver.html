restart

restart subcommand used to restart one or more VMs.

$ vermin restart vm_01
✔ Stopping vm_01
✔ Starting vm_01
✔ Establishing connection

Then you can use the command to ssh into the VM.

To restart more than one VM, pass them space separated on the command line: 

$ vermin restart vm_01 vm_02

You can also a stopped VM or a running VM.
