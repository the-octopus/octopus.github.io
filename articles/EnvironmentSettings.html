Configure Test Environment Settings

- The-Octopus-Test framework is designed to be easy to use and configurable.

- You can Configure the URL and browser type and other settings in the test project.


Steps to add Test Environment Settings:


  • Go to the folder resources / TestData .
  • Open the file ControlFile.xlsx.
  • Open the Environment Sheet.
  • Add the settings you need as follow.

    • - RunEnvironment : can takes values WEB / MOBILE .

    - RunType : can takes values Execution / Debug .

    - ProjectName : your project name .

    - Explorer : can takes values CHROME / IE / FIREFOX .

    - URL : http://your_url .

  • Follow the ControlFile.xlsx in the The-Octopus-Sample-Test-Project
