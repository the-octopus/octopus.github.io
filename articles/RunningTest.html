Running the Test Project

- The-Octopus-Test framework is python project so we have to follow the same configurations required to run a python project.

- We should make sure that the python is configured and system path variable has the python executable.

- Make sure to follow the The-Octopus-Sample-Test-Project to design your test project


Steps to run a test project :


  • Open the CMD.
  • CD to the project main folder.
  • run the bellow command.
    •          

            python main.py

    - The system will run the test and generate a HTML report with date and time stamp name as per the bellow example.

    > Reports/Automation_Result_timestamp/Automation_Report_projectName.html
  • Follow the The-Octopus-Sample-Test-Project
