Control the Test Execution

- The-Octopus-Test framework is designed based on the KeyWord approach and easy to use configurable design.

- In order to have the KeyWord feature, You need to have control the execution of defferent test scenario methods.

- Some times you do not have a need to execute all test scenario methods.

- you may run the scenario methods in defferent collections or groups.


Steps to the Execution of Scenario Methods:


  • Go to the folder resources / TestData .
  • Open the file ControlFile.xlsx.
  • Open the Driver Sheet.
  • Select the method rows in the Driver Sheet.
  • Control the execution by select the value Yes or No from the column Execution_Flag.
  • Run the test project.
  • Follow the ControlFile.xlsx in the The-Octopus-Sample-Test-Project
